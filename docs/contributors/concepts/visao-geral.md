# Visão Geral: Conceitos Principais

O **CodeGen** é construído em torno de **4 conceitos principais** que trabalham em conjunto para criar um assistente de desenvolvimento inteligente:

-  **Task (Tarefa)** - Unidade atômica de trabalho do usuário
-  **Agent (Maestro)** - Inteligência artificial orquestradora
-  **Spec-Driven Development** - Desenvolvimento orientado a especificações
-  **Session (Contexto)** - Memória e estado da conversa

:::info Benefícios da Arquitetura
Cada conceito tem uma **responsabilidade clara** e **interfaces bem definidas**. Isso permite evolução independente e facilita testes e manutenção.
:::
